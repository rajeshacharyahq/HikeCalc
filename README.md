# Salary Hike Calculator PWA

A simple Progressive Web App for calculating salary hikes.

## Features

- Enter current salary and hike percentage to calculate updated salary.
- Enter current salary and updated salary to calculate hike percentage.
- Works offline after the first visit.
- Ready for GitHub Pages hosting.

## GitHub Pages Hosting

1. Push these files to a GitHub repository.
2. Open the repository on GitHub.
3. Go to `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select your branch, usually `main`, and the `/root` folder.
6. Save and wait for GitHub to publish the site.

Your app URL will look like:

```text
https://rajeshacharyahq.github.io/HikeCalc/
```
